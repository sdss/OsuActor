Commands¶

The lvmieb replies to the follwing commands.

lvmieb¶

lvmieb [OPTIONS] COMMAND [ARGS]...

hartmann¶

control the hartmann door.

lvmieb hartmann [OPTIONS] COMMAND [ARGS]...

close¶

close the hartmann

lvmieb hartmann close [OPTIONS]

Options

-s, --side <side>¶

all, right, or left

Options

all | right | left

home¶

lvmieb hartmann home [OPTIONS]

init¶

lvmieb hartmann init [OPTIONS]

open¶

open the hartmann

lvmieb hartmann open [OPTIONS]

Options

-s, --side <side>¶

all, right, or left

Options

all | right | left

status¶

lvmieb hartmann status [OPTIONS]

help¶

Shows the help.

lvmieb help [OPTIONS] [PARSER_COMMAND]

Arguments

PARSER_COMMAND¶

Optional argument

ping¶

Pings the actor.

lvmieb ping [OPTIONS]

shutter¶

control the shutter.

lvmieb shutter [OPTIONS] COMMAND [ARGS]...

close¶

close the shutter

lvmieb shutter close [OPTIONS]

init¶

initialize the shutter

lvmieb shutter init [OPTIONS]

open¶

open the shutter

lvmieb shutter open [OPTIONS]

status¶

lvmieb shutter status [OPTIONS]

transducer¶

control the wago IOModule.

lvmieb transducer [OPTIONS] COMMAND [ARGS]...

status¶

Returns the status of transducer.

lvmieb transducer status [OPTIONS]

version¶

Reports the version.

lvmieb version [OPTIONS]

wago¶

control the wago IOModule.

lvmieb wago [OPTIONS] COMMAND [ARGS]...

getpower¶

Returns the status of wago sensor.

lvmieb wago getpower [OPTIONS]

setpower¶

Returns the status of wago sensor.

lvmieb wago setpower [OPTIONS] {shutter|hartmann_left|hartmann_right}

Options

--on¶

Required Turn on device

--off¶

Required Turn off device

Arguments

DEVICE¶

Required argument

status¶

Returns the status of wago sensor.

lvmieb wago status [OPTIONS]
