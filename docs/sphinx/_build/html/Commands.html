Commands¶

The osuactor replies to the follwing commands.

osuactor¶

osuactor [OPTIONS] COMMAND [ARGS]...

help¶

Shows the help.

osuactor help [OPTIONS] [PARSER_COMMAND]

Arguments

PARSER_COMMAND¶

Optional argument

ping¶

Pings the actor.

osuactor ping [OPTIONS]

shutter¶

control the shutter.

osuactor shutter [OPTIONS] COMMAND [ARGS]...

close¶

close the shutter

osuactor shutter close [OPTIONS]

connect¶

open the connection with the shutter.

osuactor shutter connect [OPTIONS]

disconnect¶

close the connection with the shutter

osuactor shutter disconnect [OPTIONS]

open¶

open the shutter

osuactor shutter open [OPTIONS]

status¶

osuactor shutter status [OPTIONS]

telemetry¶

Returns the status of wago sensor.

osuactor telemetry [OPTIONS]

version¶

Reports the version.

osuactor version [OPTIONS]
